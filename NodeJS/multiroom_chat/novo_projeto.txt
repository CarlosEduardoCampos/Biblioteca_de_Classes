1) Criar um novo dirétorio

2) Iniciar o NPM
    -> npm init

3) Baixar o framework
    -> npm install express --save

4) Engine de views
    -> npm install ejs --save

5) Autoloader
    -> npm install consign --save

6) Middleware
    -> npm install body-parser --save

7) Modulo que valida informações digitadas
    -> npm install express-validator --save
